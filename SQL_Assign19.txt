1) Create a view that shows all of the customers who have the highest ratings.
Ans-:	create view customer_rating
	as
	select *from customers
		where rating =any(select max(rating) from customers);

2) Create a view that shows the number of salespeople in each city.
Ans -:	create view s4
	as
	select city,count(*)from salespeople
		group by city;

3) Create a view that shows the average and total orders for each salesperson after
	his or her name. Assume all names are unique.
Ans-:	create view sp1
	as
	select sname name,count(sname) total,avg(amt) average from salespeople s,orders o
		where s.snum=o.snum
			group by sname;

4) Create a view that shows each salesperson with multiple customers.
Ans -:select sname,cname from  salespeople s,customers c
	where s.snum=c.snum
		order by sname;
